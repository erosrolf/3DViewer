School 21 programming school educational project.~\newline


The 3D viewer program a robust platform for working with 3D wireframe models. It offers essantial manipulation tools, a user-\/friendly interface, and additional customization options for an enriched user experience.~\newline


 
\begin{DoxyImage}
\includegraphics[width=800px]{open_file.gif}
\doxyfigcaption{Open file presentation}
\end{DoxyImage}
   

You can also move, zoom out, zoom in and rotate the model.~\newline
 The program is optimized to handle models with varying complexities, from 100 to 1,000,000 vertices, ensuring smooth performance without interface freezing.~\newline

\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{index_autotoc_md1}
To install, you will need Qt version 6, cmake, make, and a GCC compiler.~\newline
 Navigate to the {\ttfamily src} directory and run the {\ttfamily make install} command.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{For developers}{For developers}}\label{index_autotoc_md2}
The project is developed following the MVP pattern on the Qt framework, in C++17 language .~\newline
 The render window uses the open\+GL library.~\newline
 All code related to the Model block is documented. To view the doxygen documentation, navigate to the {\ttfamily src} directory and run the {\ttfamily make dvi} command.~\newline
 All code related to the Model block is covered on 100\% with unit tests. Run command {\ttfamily make gcov\+\_\+report} from {\ttfamily src} directory, to view the coverage report.~\newline
